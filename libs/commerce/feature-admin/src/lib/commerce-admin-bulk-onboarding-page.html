Bulk onboarding

Zoek meerdere missende sets en laat de intake daarna in één run doorlopen.

Dit is een dunne operatorflow bovenop de bestaande onboarding-engine: zoek sets in Rebrickable, kies je batch, start de run, en volg daarna direct import, seeds, sync en coverage-uitkomst.
